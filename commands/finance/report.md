## Format

Structure the financial report as follows:

1. **REPORT COVER**
   - Report title
   - Reporting entity name
   - Reporting period
   - Date of preparation
   - Prepared by / prepared for
   - Confidentiality notice: "CONFIDENTIAL - FOR INTERNAL USE ONLY" (adjust as needed)

2. **TABLE OF CONTENTS**
   - List all sections with placeholder page references

3. **EXECUTIVE SUMMARY**
   - One-page overview of financial performance
   - Key metrics with comparison to prior period and budget/plan
   - Highlights and lowlights (3-5 each)
   - Outlook and forward-looking commentary

4. **FINANCIAL HIGHLIGHTS DASHBOARD**
   Present key metrics in a dashboard format:
   ```
   Metric              | Current Period | Prior Period | Budget  | Var to Budget | Var to Prior
   --------------------|---------------|-------------|---------|--------------|-------------
   Revenue             |               |             |         |              |
   Gross Profit        |               |             |         |              |
   Gross Margin %      |               |             |         |              |
   Operating Income    |               |             |         |              |
   Operating Margin %  |               |             |         |              |
   Net Income          |               |             |         |              |
   EBITDA              |               |             |         |              |
   Cash Position       |               |             |         |              |
   ```

5. **REVENUE ANALYSIS**
   - Revenue by segment / product line / geography
   - Revenue trends (monthly, quarterly, annually)
   - Revenue drivers and commentary on variances
   - Pipeline or backlog analysis (if applicable)
   - Recommended chart: stacked bar chart showing revenue by segment over time

6. **EXPENSE ANALYSIS**
   - Expenses by category and department
   - Fixed vs. variable cost breakdown
   - Headcount and compensation analysis
   - Variance analysis (actual vs. budget, actual vs. prior)
   - Recommended chart: waterfall chart showing budget-to-actual variance

7. **PROFITABILITY ANALYSIS**
   - Margin trends across periods
   - Profitability by segment / product line
   - Contribution margin analysis
   - Break-even analysis (if applicable)
   - Recommended chart: line chart showing margin trends

8. **CASH FLOW AND LIQUIDITY**
   - Cash flow summary (operating, investing, financing)
   - Working capital analysis
   - Days sales outstanding, days payable outstanding
   - Cash runway or burn rate (if applicable)
   - Recommended chart: waterfall chart showing cash flow components

9. **BALANCE SHEET HIGHLIGHTS**
   - Key asset and liability changes
   - Debt position and covenant compliance
   - Capital structure summary
   - Book value trends

10. **BUDGET VARIANCE ANALYSIS**
    - Summary of material variances
    - For each material variance:
      - Account or line item
      - Budget amount
      - Actual amount
      - Variance (dollar and percentage)
      - Root cause explanation
      - Corrective action (if applicable)

11. **KEY PERFORMANCE INDICATORS**
    - Financial KPIs with targets and actuals
    - Operational KPIs tied to financial outcomes
    - Trend indicators (improving, stable, declining)
    - Recommended chart: KPI scorecard with status indicators

12. **FORWARD-LOOKING COMMENTARY**
    - Updated outlook for remainder of period
    - Known upcoming events or changes
    - Risks and opportunities
    - Management actions and priorities

13. **APPENDICES**
    - Detailed financial statements
    - Supporting schedules
    - Methodology notes
    - Glossary of terms

## Formatting Guidelines

- Use consistent number formatting: commas for thousands, two decimal places for currency and percentages.
- Clearly label all currency units and reporting periods.
- Use parentheses for negative numbers: $(1,234).
- Use consistent color/symbol conventions for favorable (positive) and unfavorable (negative) variances.
- Include % change alongside absolute numbers for all comparisons.
- Right-align all numerical columns.
- Bold totals and subtotals.
- Use clear section headers and sub-headers throughout.
- Present data in tables wherever possible for clarity.
- Include chart/visualization recommendations for each major section.
