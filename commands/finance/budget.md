## Format

Structure the budget plan as follows:

1. **BUDGET OVERVIEW**
   - Budget title and fiscal period
   - Budgeting entity (company, department, project)
   - Budget type (operating, capital, project, zero-based, incremental)
   - Budget methodology (top-down, bottom-up, hybrid)
   - Key strategic priorities and objectives for the budget period
   - Budget calendar and approval process

2. **REVENUE BUDGET**

   ```
   Revenue Category        | Q1       | Q2       | Q3       | Q4       | Annual   | Prior Year | % Change
   ------------------------|----------|----------|----------|----------|----------|-----------|--------
   Product Revenue         |          |          |          |          |          |           |
     - Product Line A      |          |          |          |          |          |           |
     - Product Line B      |          |          |          |          |          |           |
   Service Revenue         |          |          |          |          |          |           |
   Recurring Revenue       |          |          |          |          |          |           |
   Other Revenue           |          |          |          |          |          |           |
   TOTAL REVENUE           |          |          |          |          |          |           |
   ```

   For each revenue line:
   - State the underlying assumptions (volume, pricing, growth rate)
   - Identify seasonality patterns
   - Note dependencies or risks

3. **EXPENSE BUDGET**

   ```

   ------------------------|----------|----------|----------|----------|----------|------------
   Equipment               |          |          |          |          |          |

   For each capital item:
   - Justification and expected ROI
   - Depreciation method and useful life
   - Impact on operating expenses

5. **BUDGET SUMMARY**
   ```
   Category                | Q1       | Q2       | Q3       | Q4       | Annual
   ------------------------|----------|----------|----------|----------|--------

6. **HEADCOUNT PLAN**
   - Current headcount by department
   - Planned hires by quarter with role and compensation
   - Planned departures or restructuring
   - Fully loaded cost per employee
   - Headcount to revenue ratio

7. **ASSUMPTIONS AND DRIVERS**
   - List all key assumptions with their basis
   - Sensitivity of budget to key assumptions
   - Historical accuracy of similar assumptions
   - External factors (inflation, exchange rates, market conditions)

8. **VARIANCE TRACKING FRAMEWORK**
   - Define materiality thresholds for variance reporting
   - Establish monthly/quarterly review cadence
   - Template for variance reporting:

   | Line Item | Budget | Actual | Variance $ | Variance % | Explanation | Action Required |
   |-----------|--------|--------|-----------|-----------|-------------|----------------|
   |           |        |        |           |           |             |                |

9. **CONTINGENCY AND RESERVES**
   - Contingency amount (typically 5-10% of total budget)
   - Conditions for accessing contingency
   - Approval authority for contingency use
   - Reserve for known but unquantified risks

10. **APPROVAL AND GOVERNANCE**
    - Budget owner and approvers
    - Approval timeline
    - Amendment and reforecast procedures
    - Spending authority levels

## Formatting Guidelines

- Use consistent number formatting: commas for thousands, two decimal places for currency.
- Clearly label all currency units and periods.
- Use parentheses for negative numbers: $(1,234).
- Classify all expenses as Fixed, Variable, or Semi-Variable.
- Include prior year actuals for comparison.
- Calculate percentage of total for each line item.
- Right-align all numerical columns.
- Bold totals and subtotals.

Actual budget allocations should reflect organizational priorities, available resources, and business conditions as determined by management.
